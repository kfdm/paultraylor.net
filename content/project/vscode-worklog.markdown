A VSCode [extension] for managing a worklog in [hugo].
Inspired by [Hugo Helper] but with other functions that fit my workflow better.

## Commands

- Draft new post
- Publish Draft
- Easily tag file
- Easily copy as link
- Quick open `Today`
- Quick open `Weeklog`
